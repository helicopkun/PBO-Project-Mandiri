# PBO-Project-Mandiri
Are you a complete masochist who likes playing Soulslike or Bullet-hell?

HOW TO PLAY
W or Space = Jump or go up
A = Left
D = Right
S = Quick-fall

K = Attack
L = Parry
LSHIFT = Phase

Objective - Kill bosses, or smthng idk

Customization: 
    Arena; 
    Boss(s); 
    Boss.pattern.phase 

you can add as many bosses as you want
or as many platform as you want
in the same Game loop
